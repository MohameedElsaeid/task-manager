
## about the project
in this project I used
- php8.0
- laravel 9.0
- mysql 8.0
- nginx
- composer

used for architecture
- SAO
- repository design pattern

the project has 3 main pages

- name `create task`
- url `http://task-manager.test/tasks/cerate`
- method `POST`
-----------------------------------------
- name `users statistics`
- url `http://task-manager.test/statistics`
